# Street Coverage Planner

Mobile-first prototype for planning a missing person search route. Draw a
search area on the map, generate a coverage route, and review turn-by-turn
directions.

## Features

- Draw a custom polygon search area on a map
- Generate a sweep-style coverage route within the boundary
- View estimated distance, time, and step-by-step directions
- Optimized layout for phones with a desktop side panel

## Run locally

This is a static web app. Open `index.html` directly or run a small server:

```bash
python3 -m http.server
```

Then visit `http://localhost:8000`.
